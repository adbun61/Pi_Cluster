# Homelab Kubernetes Cluster

> ⚠️ **Work in Progress**  
> This cluster setup and GitOps workflow is under active development. Expect changes to folder structure, manifests, and automation scripts.

---

## Overview

This repository contains configuration for a personal homelab Kubernetes cluster.  
The goal is to have **reproducible, declarative deployments** for apps using:

- Kubernetes namespaces for isolation
- Kustomize for managing manifests
- GitOps workflow for deployment
- Ansible for bootstrapping the cluster and base infrastructure
